Differences between HTML and HTML5

HTML HTML5
Uses presentational elements like <font>, <center>, etc. Introduces semantic elements like <header>, <footer>, <article> for better structure and meaning.
Relies on third-party plugins (e.g., Flash) for multimedia support. Offers native support for audio and video elements, reducing dependency on external plugins.
Doesn't provide built-in support for offline web applications. Enables offline web apps using features like Application Cache and Web Storage for local data storage.
